# Menus

Este proyecto utiliza Laravel y Blade para crear un menú con varias secciones y vistas. Cada sección del menú muestra un contenido diferente, y se resalta la sección actual en el menú en función de la página en la que se encuentre el usuario.

## Requisitos

- PHP >= 7.3
- Composer
- XAMPP, WAMP, MAMP, LAMP o Laragon Stack instalado en tu sistema
